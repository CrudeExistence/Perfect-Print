Let's break down the code used on this application



*_ HTML _*

The building blocks of the site which include semantic tags and the skeleton of the website. This includes the containers, placement for everything, buttons, and more.


*_ CSS _*

Styling through CSS allows for the site to be colored, shadowed, made see-through, moved, pushed, shaped and more. Using Flexbot I was able to align all the content and make it flow better.


*_ JavaScript _*

JavaScript is where most of the magic happens. Javascript uses Axios to make requests between the server side and the front-end to fulfill requests. The first you'll notice is that the items populated were pulled with PostgressSQL from a hosted database. Then the information is looped over to create HTML elements throughout the page. It's used to apply, or remove, classes from HTML objects and displays. Through applying event listeners to the different buttons, they are programmed to perform as expected.
